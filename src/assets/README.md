# Project Images

This directory contains placeholder images for the portfolio projects.

## Required Images:
- `ecommerce.jpg` - E-Commerce Platform screenshot
- `taskmanager.jpg` - Task Management App screenshot  
- `weather.jpg` - Weather Dashboard screenshot
- `analytics.jpg` - Social Media Analytics screenshot
- `banking.jpg` - Mobile Banking App screenshot
- `ai-generator.jpg` - AI Content Generator screenshot

## Image Specifications:
- Recommended size: 800x600px or similar aspect ratio
- Format: JPG, PNG, or WebP
- Keep file sizes reasonable for web performance

## Note:
If images are not available, the component will automatically fall back to emoji icons.

## Location:
Place images in the `public/assets/` directory for them to be served correctly by the React app.
